## Agent Name Extraction

The tool extracts agent names from filename patterns:

**Supported patterns:**
- `agent-YYYY-MM-DD.md` (e.g., `marcus-2026-02-17.md`)
- `agent-output.md` (e.g., `seneca-output.md`)
- `agent-daily.md` (e.g., `galen-daily.md`)
- `agent-summary.md` (e.g., `marcus-summary.md`)
- `agent-*.md` (e.g., `argus-report.md`)

**Default:** "Unknown" if no pattern matches

## Metrics

### Word Count
- Space-separated tokens
- Approximate measure
- Good for comparing relative output

### Character Count
- Total characters in file
- More precise than word count
- Useful for storage/length metrics

### Line Count
- Number of lines in file
- Measures document structure
- Good for file complexity

### Productivity Ranking
- Average words per file
- Normalizes for output frequency
- Shows efficiency vs quantity

## Best Practices

### For Filename Conventions

Use consistent naming:

```bash
# Good patterns:
marcus-2026-02-17.md
galen-output.md
seneca-daily.md

# Avoid:
random-name.md
output-1.md
post.md
```

### For Productivity Analysis

```bash
# Look at trends, not single days
squad-stats --days 30

# Compare similar periods
squad-stats --days 7 --save # This week
# ... next week ...
squad-stats --days 7 --save # Next week

# Correlate with squad changes
# Did new tools improve productivity?
# Did squad size affect output?
```

## Troubleshooting

### No Data Found

```bash
# Check outputs directory
ls -la ~/.openclaw/workspace/outputs/

# Check file modification times
find ~/.openclaw/workspace/outputs/ -mtime -7

# Increase days
squad-stats --days 60
```

### Unknown Agent Names

```bash
# Check filenames
ls ~/.openclaw/workspace/outputs/*.md

# Rename to follow patterns
mv random-name.md marcus-2026-02-17.md
```

### Inaccurate Word Counts

```bash
# Word counts are approximate
# Use character counts for precision
squad-stats --format json | jq '.summary.Marcus.characters'

# Or use different metric (lines, chars)
```
