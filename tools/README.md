## Usage

```bash
# See all tools
squad-overview --section tools

# See squad status
squad-overview --section status

# See recent learnings
squad-learnings --days 1
```

## Getting Started

All tools are symlinked to `~/.local/bin/` for easy access:
```bash
# Use any tool
paper-summarizer 2401.12345
squad-eval --archimedes
blog-assistant --topic "AI in 2026"
squad-learnings --days 7
squad-overview
```
