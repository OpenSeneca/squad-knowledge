# dns — DNS Lookup Tool

Query DNS records (A, AAAA, MX, NS, TXT, CNAME, SOA).

## GitHub Repository

This tool will be published to: https://github.com/OpenSeneca/dns

## Installation

```bash
pip install dns-cli-tool
```

Or symlink:

```bash
ln -s $(pwd)/dns.py ~/.local/bin/dns
```

## Quick Start

```bash
dns example.com              # Query A record
dns example.com -t MX        # Query MX records
dns example.com --all        # Query all records
dns 8.8.8.8 --reverse       # Reverse lookup
```
