# claude-code-analyzer

Analyze Claude Code session logs and identify inefficiencies.

## Overview

This is an OpenSeneca tool that provides actionable insights for Claude Code sessions. It detects anti-patterns like duplicate file reads, excessive context compactions, and provides clear recommendations.

## Installation

```bash
pip install claude-code-analyzer
```

Or symlink from cloned repo:

```bash
ln -s $(pwd)/claude-analyzer.py ~/.local/bin/claude-analyzer
```

## Quick Start

```bash
# Analyze latest session
claude-analyzer

# List available sessions
claude-analyzer --list

# Detailed analysis
claude-analyzer --detailed
```
