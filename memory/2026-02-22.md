### Built: squad-dashboard-data CLI Tool

**Why:** Argus's `squad-dashboard-data.sh` generates invalid JSON (alerts field uses `{}` instead of `[]`, trailing comma). Justin needs valid data for squad dashboard.

**What:**
- **squad-dashboard-data** - Python CLI tool that generates valid JSON for squad dashboard (228 lines)

**Features:**
- **Valid JSON Output:** Always uses arrays (`[]`), never empty objects (`{}`)
- **No Trailing Commas:** Proper JSON syntax
- **Default Agent Config:** Built-in squad agents (Seneca, Marcus, Galen, Archimedes, Argus)
- **Custom Agents:** Load agents from JSON file
- **Status Checking:** Simple ping-based status detection (active/inactive)
- **Activity Tracking:** Basic activity scoring (0-100 scale)
- **Alerts Generation:** Identifies:
  - Inactive agents that should be active (research/ops roles)
  - Agents with low activity
  - Inactive ops agents (error level)
- **Validation Mode:** Verify JSON syntax before writing file

**Fixed Issues from Argus's Script:**
1. alerts field: `[]` instead of `{}`
2. No trailing comma after last agent in agents array

**Commands:**
```bash
# Generate with default agents
squad-dashboard-data --output data.json

# Generate with custom agents file
squad-dashboard-data --agents-file agents.json --output data.json

# Generate without alerts
squad-dashboard-data --no-alerts --output data.json

# Validate JSON only (don't write file)
squad-dashboard-data --validate

# Pretty-print output
squad-dashboard-data --pretty
```

**Usage for Squad Dashboard:**
