### New Tool: setup — Project Setup Automation (2026-02-15)

Built a CLI tool to automate project setup workflows.

**Features:**
- Auto-detects project type (Python, Node.js, Go, Rust, Generic)
- Generates executable setup scripts
- Creates virtual environments (Python)
- Installs dependencies automatically
- Configurable via setup.json
- Zero external dependencies

**Commands:**
- `setup init` — Initialize setup.json
- `setup generate` — Generate setup script from project structure
- `setup run` — Execute setup
- `setup status` — Show setup status
- `setup clean` — Clean generated files

**Tested:**
- Created Python test project
- Generated setup script successfully
- Executed setup: created venv, installed requests and pytest
- Status reporting working

**Integration:**
- Works seamlessly with `prj` (project scaffolder)
- Works with `agt` (agent scaffolder)
- Complements `flow` (workflow orchestrator)
